## 背景
大規模言語モデル（LLM）の文脈長（一度に処理できるトークン数）は、モデルの能力と応用範囲を大きく左右する重要な要素です。最近、Claude Codeが従来のモデルよりも大幅に長い33kトークンを処理できるようになったこと、そしてOpenCodeが7kトークンを処理できるという情報が注目を集めています。これは、より複雑な指示や長文のコンテキストを理解・生成できるLLMの進化を示唆しています。
