## 4. これだけ知っていれば大丈夫！Pythonの「超きほん」

このセクションで分かること: Pythonを動かす上で最低限知っておきたい、いくつかの基本的なルールを学べます。

プログラミング言語といっても、最初に覚えることはごくわずかです。今回は、これだけは知っておいてほしい「超きほん」を3つだけご紹介します。

### その1: 画面に文字を表示する「print関数」

コンピューターに何かをさせる第一歩は、「表示して！」と命令することです。Pythonでは、`print()`（プリント・カッコ）という命令を使います。

```python
print("こんにちは、Python！")
```

これは「**『こんにちは、Python！』という文字を画面に表示してね**」という意味です。`print`は「印刷する」という意味ですが、プログラミングでは「画面に出力する」という意味で使われます。`()`の中に表示したいものを書くのがルールです。

### その2: データを入れる「変数」

「変数（へんすう）」とは、**データを入れるための「箱」や「メモ帳」**のようなものです。たとえば、名前や数字など、一時的に覚えておきたい情報を入れておくのに使います。

```python
name = "太郎"
print(name)
```

1行目の`name = "太郎"`は、「**『太郎』という文字を、『name』という名前の箱に入れてください**」という意味です。
2行目の`print(name)`は、「**『name』という箱の中身を画面に表示してね**」という意味になります。
箱の中身はいつでも変えることができるので、とても便利です。

### その3: プログラムの説明書き「コメント」

プログラムは、自分以外の人や、未来の自分が読んだときに分かりやすいように書くことが大切です。「コメント」は、**プログラムの途中に書き込む「メモ書き」**のようなものです。Pythonは、このコメントの部分を無視してプログラムを実行します。

```python
# これはプログラムの説明です
print("Hello, World!") # 画面に挨拶を表示
```

`#`（シャープ記号）を文頭につけると、その行はコメントになります。
「**これはプログラムのメモ書きで、Pythonは読み飛ばします**」という意味ですね。プログラムを分かりやすくするために、とても大切な機能です。

この3つ、どうでしたか？
「なーんだ、意外と簡単そう！」と感じていただけたら嬉しいです。大丈夫、難しくないです。まずはこの「超きほん」を頭の片隅に置いておきましょう。

## 5. 実際にPythonを動かしてみよう！コピペから始めよう

このセクションで分かること: これまで学んだ基本を使って、実際にPythonのコードを動かす体験ができます。

さあ、いよいよPythonを実際に動かしてみる時間です！難しく考える必要はありません。まずは、ここにあるコードを**コピー＆ペースト（コピペ）**して、動かしてみましょう。

今回は、先ほど紹介した「Google Colaboratory」を例に説明しますが、他のオンライン実行環境でも基本的な操作は同じです。

1.  **Google Colaboratoryにアクセス**:
    Googleアカウントがあれば無料で使えます。`https://colab.research.google.com/` にアクセスし、「ノートブックを新規作成」をクリックします。
2.  **コードを貼り付ける**:
    表示された入力欄（「セル」と呼びます）に、以下のコードをコピーして貼り付けてみましょう。

    ```python
    # 最初のPython体験！
    print("Hello, Python World!")

    # あなたの名前で挨拶してみよう
    my_name = "あなたの名前" # ここをあなたの名前に変えてみましょう！
    print("こんにちは、" + my_name + "さん！")
    ```

3.  **実行する**:
    貼り付けたセルの左にある、三角形の「再生ボタン」のようなアイコン（または `Shift + Enter` キー）をクリックします。

どうでしたか？
セルの下に「Hello, Python World!」や「こんにちは、[あなたの名前]さん！」と表示されましたか？もし表示されていれば、あなたはもうPythonプログラマーの仲間入りです！

「**これはまず『Hello, Python World!』と表示し、次に『あなたの名前』を『my_name』という箱に入れて、それを使って挨拶のメッセージを表示しています**」という意味です。

まずはコピペで動かして、その結果を見るだけでも十分楽しいですよ！
もし「あなたの名前」の部分を自分の名前に変えてもう一度実行したら、どうなるでしょうか？ぜひ試してみてください。小さな変化ですが、自分でプログラムを「操っている」感覚が味わえるはずです。
