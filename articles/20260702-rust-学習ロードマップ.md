# Rust 学習ロードマップ

## 1. なぜRustを学ぶか (5分)
### 学習時間: 5分
### 到達目標: Rustの主な特徴と用途を理解する

Rustは高性能なシステム開発に適したプログラミング言語です。特にメモリ管理や安全性において優れています。C++のような低レベル制御が可能な言語でありながら、安全性和堅牢性を確保できる点が大きな特徴です。

例えば、ウェブサーバーやスマートフォンアプリケーションのバックエンドなど、システム開発で重要な場面での活用例があります。またRustはコミュニティも強く、最新の技術や解決策を得やすい環境があります。

## 2. Rust入門 (1日)
### 学習時間: 1日
### 到達目標: Rustの基本的な文法と構造を理解する

まずは、Rustの基本的な文法と構造について学びます。以下の例では、基本的なプログラムを作成します。

```rust
// main.rsファイルを作成し、以下のようなコードを入力します。
fn main() {
    println!("Hello, world!");
}
```

このコードは、`main()` 関数内で `"Hello, world!"` という文字列を表示します。Rustの基本的な構造を理解することで、次に進む準備ができます。

## 3. バックグラウンド知識 (1日)
### 学習時間: 1日
### 到達目標: 開発者にとって必要なバックグラウンド知識を身につける

Rustの入門後に、いくつかのバックグラウンド知識を学びます。このセクションでは、以下の項目を理解することを目指します。

- **コンピュータ体系構造**：メモリ管理やシステムコールについて
- **コンパイラとライブラリ**：Rustが提供する機能を理解する

これらは言語の基礎となる知識であり、学び始める前にこれらの知識を把握しておくことで、後の学習をスムーズに行うことができます。

## 4. Rustの基本文法と構造 (2日)
### 学習時間: 2日
### 到達目標: Rustの基本的な機能と文法を理解し、簡単なプログラムを作成する

このセクションでは、Rustの基本的な構文と特徴について学びます。以下の例を通じて理解度を深めていきます。

```rust
// vec_example.rsファイルを作成し、以下のようなコードを入力します。
fn main() {
    let v: Vec<i32> = Vec::new();

    // ベクタに要素を追加する
    v.push(10);
    v.push(20);

    println!("v[0]: {}", v[0]);
}
```

このプログラムは、ベクトル（動的な配列）の基本操作を行います。Rustでは型推論が行われるので、型を明示的に指定しなくても大丈夫です。

## 5. ライフタイムと共有データ (2日)
### 学習時間: 2日
### 到達目標: Rustのライフタイムと共有データの管理方法を理解する

Rustは安全なプログラミングを重視しており、ライフタイムはその重要な概念の一つです。以下の例でライフタイムを理解しましょう。

```rust
// lifetimes.rsファイルを作成し、以下のようなコードを入力します。
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() {
        x
    } else {
        y
    }
}

fn main() {
    let string1 = String::from("long string is long");
    let result = longest(string1.as_str(), "short");
    println!("The longest string is {}", result);
}
```

このプログラムでは、関数`longest`が返す参照のライフタイムを指定しています。Rustは参照型を使用することで、安全な共有データ管理を実現します。

## 6. モジュールとテスト (1日)
### 学習時間: 1日
### 到達目標: Rustでモジュール化したコードとテストを理解し、実装する

Rustでは、`mod`キーワードを使用してモジュールを作成できます。これにより、プログラムの構造を整理しやすくします。また、テストは品質保証のために重要です。

```rust
// lib.rsファイルを作成し、以下のようなコードを入力します。
mod util;

pub fn run() {
    println!("util::add(1, 2) = {}", util::add(1, 2));
}

// src/util/mod.rsファイルを作成し、以下のようなコードを入力します。
pub fn add(a: i32, b: i32) -> i32 {
    a + b
}
```

この例では、`lib.rs`でモジュール`util`を使用し、その中にある関数`add`を呼び出しています。

## 7. 実装プロジェクト (2週間)
### 学習時間: 2週間
### 到達目標: Rustで小さなプロジェクトを実装する

実際にRustを使って開発を行うことで、理解した知識を実践的に学びます。プロジェクトは任意ですが、例えば命令行アプリケーションを作成することができます。

例：`cargo new calculator --bin`

```rust
// src/main.rsファイルを作成し、以下のようなコードを入力します。
use std::io;

fn main() {
    println!("Calculation Program");

    let mut input1 = String::new();
    let mut input2 = String::new();

    io::stdin().read_line(&mut input1).expect("Failed to read line");
    io::stdin().read_line(&mut input2).expect("Failed to read line");

    let num1: f64 = input1.trim().parse().expect("Please type a number!");
    let num2: f64 = input2.trim().parse().expect("Please type a number!");

    println!("The result is {}", num1 + num2);
}
```

このプログラムは、ユーザーから入力を受け取り、その数値を加算します。

## 8. 拡張と応用 (継続)
### 到達目標: Rustの最新技術やライブラリに触れ、自身のスキルレベルをさらに高める

Rustは日々進化しており、新しい特徴やツールが追加されています。本ロードマップの最後では、これらの最新情報を学び、既存の知識に組み込んでいきます。

例えば、`async/await`を使用した非同期プログラミングや、`Cargo`の効率的なプロジェクト管理機能などについて学ぶことができます。

Rustは高度なシステム開発に最適な言語であり続けているため、定期的に最新情報を追加することでスキルを向上させることができます。
