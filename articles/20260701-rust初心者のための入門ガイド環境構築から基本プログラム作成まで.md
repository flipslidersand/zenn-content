## 環境構築
Rustを始める前に、いくつかのツールをセットアップする必要があります。これらのツールは通常、Rustの最新バージョンをインストールし、プロジェクトを作成し、ビルドするためのものです。

### 必要なツール
1. **Rustup**: Rustの最新版や異なるバージョンを管理します。
2. **Cargo**: Rust用のパッケージマネージャで、依存関係の管理からビルド、テストまで一貫したワークフローを提供します。

## インストール手順
### Windows
1. **Rustup** のインストール:
   ```bash
   # PowerShell またはコマンドプロンプトで以下を実行
   Invoke-WebRequest -Uri "https://sh.rustup.rs" -OutFile rustup-init.sh
   ./rustup-init.sh --no-modify-path
   ```

2. **Cargo** のインストール:
   Rustupが自動的にCargoもインストールします。

### macOS
1. **Homebrew** を利用してRustをインストール:
   ```bash
   # Homebrewのインストール（初回のみ）
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/main/install.sh)"

   # RustとCargoのインストール
   brew install rustup
   ```

### Linux
1. **Rustup** のインストール:
   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```
   インストール時にパスを変更したい場合は、--component cargo を指定してCargoもインストールします。

## Hello, World!
Rustの基本的なプログラムである "Hello, World!" プログラムを作成しましょう。この例では、エディタで新しいファイルを作成し、コードを記述します。
```rust
// main.rs
fn main() {
    println!("Hello, World!");
}
```

### エディタの選択
RustはIDEやテキストエディタに優しい言語なので、自分の好みや開発スタイルに基づいてエディタを選択できます。以下には、人気のある選択肢を紹介します。

- **Visual Studio Code (VSCode)**: Rust插件提供，支持智能代码补全和调试。
- **IntelliJ IDEA**: 集成开发环境（IDE），适合有经验的开发者。
- **Sublime Text / VSCode + RLS插件**: 轻量级且易于使用。

## 開発環境の設定
### インテルライブラリの導入
インテルライブラリはオプションですが、パフォーマンスの向上や特定の機能を活用するために必要かもしれません。通常、Cargo.tomlファイルに必要な依存関係を追記します。

```toml
[dependencies]
# 例：OpenCVライブラリのインストール
opencv = "0.26"
```

### プロジェクト構築方法
1. **Cargo initialize**:
   ```bash
   cargo new hello_world
   cd hello_world
   ```

2. **Cargo.tomlファイル編集**（必要に応じてインデックス追加）。

3. **Cargo Run**:
   ```bash
   cargo run
   ```

## 実践的な練習例：配列とループを使ったプログラム作成
### 配列の定義と使用
```rust
// main.rs
fn main() {
    let numbers = [1, 2, 3, 4, 5];
    
    for number in numbers.iter() {
        println!("{}", number);
    }
}
```

このプログラムでは、配列に数値を格納し、forループを使って各要素を出力します。これはRustの基本的なデータ構造と制御フローを理解する良い練習です。

### ループの使用
```rust
// main.rs
fn main() {
    let mut count = 0;
    
    while count < 5 {
        println!("Count: {}", count);
        count += 1;
    }
}
```

この例では、whileループを使用して指定された回数までプログラムを実行します。これは条件の反復処理を行う方法を示しています。
